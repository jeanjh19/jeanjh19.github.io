Data Wrangling - Western Constellation, Chinese Zodiac, and Nobel Laureate

Project working with Miaomiao Niu

Project Description

The project mainly examines the relationship between Nobel laureate and western constellation, and the relationship between Nobel laureates and Chinese zodiac. Through reading the project, you can know that which western constellation has the largest number of the Nobel laureates, which western constellation has the largest number of the Nobel laureates in each Nobel category, which Chinese zodiac has the largest number of the Nobel laureates, and which Chinese zodiac has the largest number of the Nobel laureates in each Nobel category. The ETL process is conducted by Python and MySql. The analysis and result are visualized by using Tableau and Excel.
